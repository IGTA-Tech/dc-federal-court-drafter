## Format Checklist

- [ ] Font: Times New Roman, 12 point
- [ ] Spacing: Double-spaced, 2 spaces between sentences
- [ ] Margins: 1 inch minimum on all sides
- [ ] Page Numbers: Present on all pages
- [ ] Caption: Includes judge initials in case number
- [ ] Pin Cites: All citations include page numbers
- [ ] Signature Block: Complete with DC Bar number
- [ ] Certificate of Service: Included
- [ ] Page Count: Within 45-page limit (or 25 for reply)
- [ ] File Format: Text-searchable PDF
